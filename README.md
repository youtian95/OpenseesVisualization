# Opensees Visualization Tool

Opensees模型可视化：模态，变形，和结点编号
Opensees model Visualization


### 需要

 - Matlab
 
### 使用

 1. 将所有模型tcl文件拷贝到DisplayFiles文件夹下
 2. 将ExampleScript.m中InputFileName变量设置为入口tcl文件的文件名字
 3. 运行ExampleScript.m
 4. 在窗口输入任意字符并回车退出程序
